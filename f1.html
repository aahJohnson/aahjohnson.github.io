1. Beskriv vad var och en av protokollen TCP och IP gör på sändande och mottagande sida.

- IP (Internet Protocol): Delar upp data i paket och adresserar dem (anger avsändare/mottagare). På mottagarsidan används IP för att känna igen rätt adress och ta emot paketen.

- TCP (Transmission Control Protocol): Säkerställer att paketen kommer fram i rätt ordning och att förlorade paket skickas om. På mottagarsidan pusslar TCP ihop data korrekt och skickar bekräftelser till sändaren.

2. Beskriv kortfattat de steg som klienten (webbläsaren) och webbservern tillsammans går igenom...

1. Klienten (webbläsaren) skickar en förfrågan (HTTP-request) till servern om att hämta index.html.

2. Servern letar upp filen och skickar tillbaka innehållet (HTTP-response).

3. När webbläsaren läser index.html ser den referenser till style.css och example.jpg.

4. Webbläsaren skickar nya förfrågningar för dessa filer.

5. Servern svarar med CSS-filen och bildfilen.

6. Webbläsaren renderar sidan med HTML, CSS och bild.
