Vad är en regel i CSS och vad består den av?

- En regel består av en selektor (t.ex. p, .klass, #id) och en deklarationsblock med egenskaper inom { }

Vad är skillnaden mellan id och class? När bör man använda id respektive class?

- Ett id ska vara unikt och användas för ett specifikt element, exempelvis en unik titel.

- En class kan användas för element som ska dela css, exempelvis texter.
